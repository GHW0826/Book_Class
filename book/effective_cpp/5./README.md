
## 26. 변수 정의는 늦출 수 있는 데까지 늦추는 근성을 발휘하자
  
  - 생성자, 소멸자가 있는 변수 사용시, 생성자, 소멸자의 비용이 들기 때문에 사용하기 바로 전에 선언하자.
```cpp
  // 1. 생성자 1번, 소멸자 1번, 대입 n번
  TEST t;
  for (int i = 0; i < n; ++i)
    t = i;
  
  // 2. 생성자 n번, 소멸자 n번
  for (int i = 0; i < n; ++i)
    TEST t(i);
```
